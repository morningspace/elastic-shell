
Elastic Shell

Usage: elash COMMAND [OPTIONS] SUBCOMMAND [ARGS]

Commands:
  index               manage index
  snapshot            manage snapshot
  reindex             run reindex
  upgrade             run cluster upgrade

Run 'elash COMMAND --help' for more information on a command.
Run 'elash COMMAND --ui-text' to launch text UI.
Run 'elash COMMAND --ui-dialog' to launch dialog UI.
Run 'elash COMMAND --dry-run' to launch in dry run mode.
